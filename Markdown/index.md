# 3. 강조<span id="emphasize"></span>
기울임 = *별표 한개*, _언더바 한개_ 로 감싼다

두껍게 = **별표 두개**, __언더바 두개__ 로 감싼다

**_두개를 같이 사용가능_**

취소선 = ~~물결표시 두개로 감싼다~~

밑줄 = <u>u태그로 감싼다</u>

인라인강조 = `인라인강조는 억음부호로 감싼다`

# 4. 목록<span id="list"></span>

- 순서가 필요하지 않은 목록에 사용 가능한 기호
  - 대쉬(hyphen) 1개
  * 별표(asterisks) 1개
  + 더하기(plus sign) 1개
